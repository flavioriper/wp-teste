Preencha automaticamente o endereço a partir do CEP no WooCommerce

== Description ==

Este plugin vai auto completar o endereço no checkout a partir do CEP informado.

Nenhuma configuração é necessária. Basta ativar e pronto!

= Por que usar esse plugin? =

Esse plugin funciona diretamente do navegador, sem precisar de banco de dados.

Desenvolvido com uma API otimizada, que busca em múltiplos serviços ao mesmo tempo, retornando o que for mais rápido. Além de um poderoso sistema de cache que reduz o tempo de consulta para até 0.1 segundo.

Embora o plugin dos Correios do Claudio Sanches funcione bem (inclusive inspirou esse), ele pode ser lento por depender unicamente dos Correios.

Além disso, esse plugin é útil caso você não utilize os Correios, por exemplo.
